
# Manshu Learning — Premium Training Website

## Overview
Build a modern, premium, fully responsive website for "Manshu Learning" — a professional training brand. The site will feature smooth animations, clean typography (Inter/Poppins), and a polished corporate aesthetic matching the teal/gold logo colors.

## Pages & Sections

### Single-Page Layout with Smooth Navigation

1. **Navbar** — Sticky header with the Manshu Learning logo, nav links (About, Services, Why Us, Contact), and a "Get in Touch" CTA button. Mobile hamburger menu.

2. **Hero Section** — Full-width hero with heading "Industry-Relevant Learning for Real-World Success", subheading, two CTA buttons ("Explore Programs" + "Contact Us"), and an abstract gradient/geometric background using brand colors (teal + gold).

3. **Highlights Section** — 4 animated cards in a grid: Industry-relevant courses, Expert mentors, Hands-on learning, Flexible programs. Icons + hover lift effects.

4. **About Section** — "Who We Are" with the provided copy, clean two-column layout with subtle fade-in animation.

5. **Mission & Vision** — Side-by-side split cards with distinct styling for Mission and Vision statements.

6. **Why Choose Us** — 4 feature items with icons: Experienced trainers, Customized programs, Blended learning, Measurable outcomes.

7. **Services Section** — 4 premium service cards (Corporate Training, E-Learning, Skill Development, Executive Coaching) with descriptions and hover animations.

8. **Testimonials** — 3 dummy professional testimonials in a card carousel/grid with quote icons and star ratings.

9. **CTA Banner** — Bold call-to-action: "Partner with Manshu Learning to build future-ready teams" with "Get in Touch" button.

10. **Contact Section** — Functional contact form (Name, Phone, Email, Subject, Message) with validation, plus contact info sidebar (email, phone, address). Form will use EmailJS or a similar client-side email service for real submissions with success/error feedback.

11. **Footer** — Logo, quick links, contact info, copyright "© 2025 Manshu Learning. All Rights Reserved."

## Design System
- **Colors**: Teal (#2BB5A0) primary, Gold/Amber (#E8B84B) accent, dark text, light backgrounds
- **Fonts**: Inter for body, Poppins for headings (via Google Fonts)
- **Animations**: Scroll-reveal fade-ins, card hover lifts, button micro-interactions, smooth section transitions
- **Style**: Clean minimal corporate with soft shadows, subtle gradients, glassmorphism accents

## Contact Form
- Client-side validation (required fields, email format, phone format)
- Integration with EmailJS (free tier, no backend needed) to send form data to info@manshulearning.com
- Success toast notification + form reset on submit
- Error handling with user-friendly messages

## Technical
- React + Tailwind CSS + TypeScript
- Responsive across mobile, tablet, desktop
- SEO-friendly semantic HTML
- Component-based architecture
- Smooth scroll navigation between sections
